# 테스트 작성 가이드

테스트를 작성하면서 겪었던 경험을 토대로 좋은 테스트를 작성하기 위해서 생각해 보아야 할 내용을 안내합니다. 여기에 적힌 내용이 "진리"는 아니며 테스트 작성자가 처한 상황과 환경에 따라 다른 결론을 얻을 수도 있습니다. 새로운 아이디어가 생길 때마다 문서를 계속 업데이트 할 생각입니다.
